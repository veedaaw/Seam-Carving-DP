# Seam-Carving-DP
Implementation of a seam carving technique using dynamic programming and the OpenCV library.


Prerequisites:

cmake, opencv


To compile in the lab
——————————

run ./lab_config.sh

cd build

make

To compile on your own machine

———

cd build/

cmake ../

make


To run

——

In the build folder:

./sc image1 new_width new_height image2
